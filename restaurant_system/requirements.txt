# Restaurant Order Management System - Dependencies
#

# - threading: For multi-threaded programming and synchronization
# - tkinter: For GUI development (usually included with Python)
# - queue: For thread-safe communication
# - time: For timing and delays
# - random: For simulation randomness
# - dataclasses: For Order data structure
# - enum: For status enumerations

# - argparse: For command-line argument parsing
# - signal: For graceful shutdown handling
